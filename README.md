# 🐍 Игра "Змейка"

Классическая игра "Змейка", созданная в рамках практической работы по курсу "Введение в специальность".

## 🎮 Как играть

- Используйте стрелки на клавиатуре для управления змейкой
- Съедайте красные яблоки для увеличения счета
- Избегайте столкновения со стенами и собственным телом
- С каждым съеденным яблоком игра ускоряется

## 🛠️ Технологии

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API

## 📊 Особенности

- ✨ Красивый современный дизайн
- 🏆 Система рекордов (сохраняется в браузере)
- ⚡ Динамическое увеличение скорости
- 👀 Анимированная змейка с глазами
- 🎯 Плавное управление
